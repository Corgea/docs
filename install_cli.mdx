---
title: 'Install the CLI'
description: 'Install the Corgea CLI on MacOS, Windows, or Linux.'
---
<CodeGroup>
```bash MacOS
curl -L https://cli.corgea.app/aarch64_darwin/latest -o corgea.zip && unzip corgea.zip
chmod +x corgea
sudo mv corgea /usr/local/bin
```

```bash Windows
Download the latest binary from https://cli.corgea.app/x86_64_windows/latest

Move the executable to a directory in your PATH.
```

```bash Linux
curl -L https://cli.corgea.app/x86_64_linux/latest -o corgea.zip && unzip corgea.zip
chmod +x corgea
sudo mv corgea /usr/local/bin
```
</CodeGroup>


## Login with the CLI

<AccordionGroup>
    <Accordion title="Corgea.app instance" defaultOpen="true">
You can copy your Corgea token from the app.

[https://www.corgea.app/integrations/](https://www.corgea.app/integrations/)

    ```bash
    corgea login <YOUR CORGEA TOKEN>
    ```
    </Accordion>
    <Accordion title="Dedicated Instance">
    <CodeGroup>
    ```bash MacOS & Linux
    CORGEA_URL="https://your-instance.corgea.app" corgea login <YOUR CORGEA TOKEN>
    ```

    ```powershell Windows
    $env:CORGEA_URL="https://your-instance.corgea.app"
    corgea login <YOUR CORGEA TOKEN>
    ```
    </CodeGroup>
    </Accordion>
