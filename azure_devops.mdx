---
title: 'Azure Devops'
description: 'Generating pull requests in Azure Devops.'
---

<Steps>
  <Step title="Create a Azure Devops personal access token">
      Navigate to your Azure Devops account and create a personal access token.
    <Frame>
      <img src="/images/azure-devops-1.png" style={{ borderRadius: '0.5rem' }} />
    </Frame>
    The token should have read and write permissions.
    <Frame>
      <img src="/images/azure-devops-2.png" style={{ borderRadius: '0.5rem' }} />
    </Frame>
  </Step>
  <Step title="Add the token to Corgea">
      Go to the <a href="https://www.corgea.app/integrations/" target="_blank">integrations</a> page in Corgea and paste the token in the Azure Devops section.
        <Frame>
          <img src="/images/azure-devops-3.png" style={{ borderRadius: '0.5rem' }} />
        </Frame>
  </Step>
  <Step title="Issue a PR to Azure Devops">
      Now you can issue a PR from the issue in Corgea.
    <Frame>
      <img src="/images/azure-devops-4.png" style={{ borderRadius: '0.5rem' }} />
    </Frame>
  </Step>
</Steps>
